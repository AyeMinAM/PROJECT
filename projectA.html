Travellers' Journey

A card design is one of the most significant visual elements of the Kootour website. This is because a user who visits the home page, along with certain other pages of the site, will see the cards first. Each card is a product and a service that Kootour is offering, so I carefully designed each card with the most important features of the product, such as: price, name of the tour, name of a guide, and a brief description about the tour. An appealing image of the tour activities was used to capture the attention of users. On the other hand, I also familiarized myself with the Bootstrap card component so that the front end developer could easily implement my design.

Once the potential users click on the card, they can see the full details of the trip. Information about the trip is mentioned in the order of importance. I made sure that the most important information of the trip was always located above the fold of the screen. Bootstrap's 12 column grid is used effectively to design the layout and the contents so that they all display all well across different screen sizes.

Icons Colour Palette

Icons color palette is one the most important visual features of Kootour. The icon color palette is produced to cordinate well with the primarary color of 'Kootour Turquoise' , and different spectrum of grey colors. 'Kootour Turquoise' is mainly used for the single color icons and different color of the palette is used in some of the icons to make the site compelling.

Colour of the Social Media Icons

The colours of the social media icons are the official colours from each respective social media.

Typeface

I use the Montserrat Google font created by Julieta Ulanovsky for the Kootour project because it is open source and features a modern geometric font look which is appealing on a travel site. Monsterrat Light is used for titles, subtitles and paragraphs, while Monserrat Medium is used for headers, prices and the button texts, and any important text that needs to be hilighted.
